## 📝 项目简介

**RealLifeGalGame** 是一个基于 **Unreal Engine 5.6.1** 和 **Oculus Quest 3** 的 **MR 恋爱模拟游戏**。

玩家将通过 MR 技术，在真实世界里与虚拟角色自然互动。游戏的核心特色是在对话过程中实时弹出 **对话框与选项**，将 Galgame 的体验无缝融入现实世界。



## 🕹️ 核心特性

- 🎭 **现实世界中的对话系统**  
  渲染经典 Galgame 对话框与选项 UI，打破虚拟与现实的边界。

- 💬 **AI 驱动的多分支对话**  
  结合结构化 JSON 数据与 AI 模型，实时理解玩家发言，动态生成三条风格各异的男主回应。

- ✋ **自然手势交互**  
  支持 Oculus Quest 3 手部追踪，通过直观手势选择对话或操作 UI，增强沉浸感。

- 🌐 **深度混合现实体验**  
  将真实环境、虚拟角色与游戏逻辑融合，让玩家在熟悉的现实空间体验虚拟恋爱。

## 🚀 快速开始

### 依赖环境
- Unreal Engine 5.6.1
- Oculus Quest 3 开发者模式开启
- Meta Quest Link 连接 PC 调试

### 安装与运行
1. 下载项目工程：[📦 Google Drive](https://drive.google.com/file/d/1s8FLfAOd6z-uRuKdebwoEULcY6Kzh74n/view?usp=sharing)  
2. 连接 Quest 3，打开 UE 工程并运行 VR 模式，即可进入 MR 场景。
